Про мене

Мій шлях в розробку розпочався з знайомства з такою програмою як excel. Протягом семи років я займався фінансами на фірмі яка займалась ремонтними роботами приміщень. В процесі роботи я зіткнувся з потребою автоматизації свого обліку. Я почав будувати систему обліку з допомогою програми excel. Протягом своєї роботи я постійно вдосконалював свою облікову систему, придумував нові функції, виводив нові показники. Процес розробки почав приносити мені більше задоволення ніж моя основна діяльність.

Зрозумівши що розробка мені найбільше імпонує ніж звичайна рутинна робота я почав цікавитись різноманітними технологіями. Спочатку я переглянув кілька відео в YouTube де розповідалось про основи Java. Я почав цікавитись можливостями цієї мови програмування і навіть планував розпочати її вивчення. Але якось я натрапив на безкоштовний вебінар GO IT де розповідалось про створення сайтів та різних веб-застосунків. Мене зацікавила ця тема і в листопаді 2021 року я розпочав навчання на курсі fullstack в GO IT.

Протягом навчання ми реалізували багато цікавих проектів як на ванільному Java Script так і на React. Останній командний проект, який ми реалізовували з командою випускників був з використанням такої технології як React Native.

Технології я якими я працюю це - HTML, CSS, SASS, Java Script, React та React Native.
